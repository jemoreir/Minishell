# Minishell

A minimal Unix shell implementation developed as part of the 42 Lisboa curriculum.

## Status

🚧 Work in progress

This project is currently being developed collaboratively.  
The repository is temporarily separated into two parts while the implementation is being developed individually before merging into a unified version.

At the current stage, the main focus is on parsing and shell structure.  
Execution features are still being integrated.

## Project Goals

The goal of this project is to build a minimal Unix shell capable of parsing commands and handling common shell features.

## Concepts Used

- command parsing
- tokenization
- pipes and redirections
- environment variables
- signal handling
- file descriptors
- process creation (fork, execve)

## Language

- C

## Notes

This project is still under development, so some features may be incomplete or subject to change.
