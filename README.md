# Minishell

Implementation of a simple Unix shell developed at 42 Lisboa.

## Status

🚧 Work in progress

This project is currently being developed collaboratively.
The repository is temporarily separated into two parts while
the features are implemented individually before merging.

## Project Goals

The goal of this project is to implement a minimal Unix shell
capable of executing commands and handling common shell features.

## Concepts Used

- process creation (fork)
- pipes
- file descriptors
- command parsing
- environment variables
- signals

## Language

C
